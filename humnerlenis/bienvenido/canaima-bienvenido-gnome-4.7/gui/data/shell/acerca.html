Arceca de Canaima GNU/Linux

Una distribución GNU/Linux basada en Debian que surge como una solución, en principio, para cubrir las necesidades ofimáticas y de escritorio de los usuarios finales de la Administración Pública Nacional.

Versión 4.0

Totalmente en Software Libre

Realizada en Venezuela por talento nacional.

Hemos desarrollado un sistema de software libre para garantizar la libertad de los usuarios.

Versión 4.0

Algunas Ventajas

Canaima GNU/Linux posee aplicaciones diversas para llevar a cabo tareas de ofimática, internet, gráficos, entretenimiento, desarrollo y configuración de todo el sistema.
